I am a co-developer on a web app, with Frankie Logan and Emily Smith. The web app is an interactive map that shows riparian vegetation along the Rio Grande Corridor in New Mexico. This project seeks to create a timeline where map user's can scroll through 30 years of riparian vegetation cover. We will create a map overlay function showing percipitation levels. We hope to include addtional graphs and supplemental information, such as yearly Carbon Emissions.
